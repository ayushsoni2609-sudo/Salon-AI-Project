# Project Overview
This project is an AI-Powered Salon Automation & Recommendation System developed as part of my BCA Final Year. It helps salon owners automate their operations and provides data-driven insights to grow their business.

# Key Features

AI Recommendation Engine: Suggests the best services to customers based on their visit history using Cosine Similarity.

Business Dashboard: Visualizes top-performing services and customer demographics using interactive charts.

Automated Marketing: Generates personalized SMS templates for customer engagement.

User-Friendly Interface: Built with Streamlit for a fast and responsive web experience.

# Tech Stack

Language: Python

Data Science: Pandas, NumPy, Scikit-Learn

Web Framework: Streamlit

Dataset: 10,000+ Customer Records
